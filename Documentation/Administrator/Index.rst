Installation
------------

Installation is pretty straightforward. Either install the extension from the TYPO3 Extension Repository (TER) or clone it directly from the official GitHub repository at https://github.com/zechendorf/easyquiz.

Do not forget to add the static TypoScript template (Easy Quiz) to your website's template.
